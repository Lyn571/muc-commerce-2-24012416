## 用自己的话回答

- 什么是特征，什么是标签：特征是模型进行判断时可以查看的用户信息，例如使用月数、投诉情况和消费金额；标签是希望模型预测的真实答案，本项目的标签是Churn。
- 为什么要保留测试集：测试集不参与模型学习，用来模拟模型从未见过的新用户，检查模型是否真正具备预测能力，而不是只记住训练数据。
- 为什么83%准确率仍可能没有用：因为大约83%的用户本来就没有流失，即使模型把所有用户都预测为未流失，也能得到约83%的准确率，但流失召回率为0，无法找到任何真正需要挽留的用户。
